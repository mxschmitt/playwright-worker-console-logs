Worker Types Demo

This demo shows how to log complex objects to the console in all three types of workers. Open your browser's developer console to see the logged objects.

1. Web Worker

A dedicated worker that runs in the background, separate from the main thread.

Click the button to test Web Worker

2. Service Worker

A worker that acts as a proxy between your app and network, enabling offline functionality.

Click the button to test Service Worker

3. Shared Worker

A worker that can be accessed by multiple scripts running in different windows/tabs.

Click the button to test Shared Worker
